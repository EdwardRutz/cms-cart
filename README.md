# CMS Cart

- A demo of an Express-Node JS site.

## DEPENDENCIES

- Express JS
- [Embedded JavaScript Templating (EJS)](https://ejs.co/)
- [nodemon](https://nodemon.io/)


## INSTALL & RUN

- To run MongoDB as a Mac service, `brew services start mongodb-community@4.2`
